Security Audit #047

Status: Recovered | Target: E-commerce SME

The Incident

Server was compromised via a brute-force attack on a weak administrator password, leading to temporary site defacement.

The Diagnosis

Audit logs showed 5,000+ failed SSH login attempts from foreign IPs before the successful breach via password guessing.

The Hardening

Wiped and restored system from clean backups. Disabled password authentication entirely in favor of Ed25519 SSH Key-Based Auth.

The Outcome

Brute-force attack surface eliminated; server integrity restored; implemented Fail2Ban to automatically drop malicious IPs.
